Bem-vindo à Loja de Peças - Sistema de Gerenciamento

Este sistema de gerenciamento de peças é projetado para fornecer uma solução eficiente para cadastrar, consultar e remover peças em uma loja. Aqui estão algumas das principais funcionalidades:

  • Cadastrar novas peças com informações detalhadas.
  • Consultar informações sobre peças cadastradas, pesquisando por código ou fabricante.
  • Remover peças do banco de dados de forma segura e eficiente.
  • Cadastrar novos clientes com informações completas para melhor atendimento.
